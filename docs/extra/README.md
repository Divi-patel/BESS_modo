## Key Insights (from EIA analysis)

- **1,331 BESS units** in US fleet (882 operating as of 2024)
- **48%** of BESS capacity is co-located with solar generation
- Only **8%** of BESS units have pricing node data in EIA Form 860 — a critical gap for nodal revenue analysis
- ERCOT leads with **155 operating units** totaling 10,105 MW
- LFP (lithium iron phosphate) is the dominant chemistry for new builds
