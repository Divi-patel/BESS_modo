# Data Directory

## Structure

| Path | Description | Tracked |
|------|-------------|---------|
| `bess_enriched.parquet` | 1,331 US BESS units from EIA-860, enriched with coordinates, pricing nodes, chemistry, coupling status | Yes |
| `engineering__bess_enriched.md` | Schema documentation for the BESS enriched dataset | Yes |
| `results/` | Backtest outputs — summary CSVs, sensitivity analysis, chart PNGs | Yes |
| `prices/` | ERCOT LMP parquets — 4 hubs + 5 resource nodes, RT + DA hourly (2010-2025) | No (re-downloadable) |
| `revenue/` | Historical generation revenue by site — 6 ERCOT solar/wind assets | No (re-downloadable) |
| `generation/` | Solar/wind generation data — 6 ERCOT assets, hourly | No (re-downloadable) |
| `extra/` | Schema docs, pricing node coordinates, pipeline documentation | No (proprietary) |

## Reproducing the Data

Large data files are excluded from version control but can be regenerated by running `notebooks/01_data_download.ipynb`, which downloads from the cloud data pipeline. Authentication is required.

As a fallback, ERCOT price data can be obtained directly using the [gridstatus](https://github.com/kmax12/gridstatus) Python library.

## Key Datasets

### BESS Fleet (`bess_enriched.parquet`)
- **Source:** EIA Form 860 (2024 release)
- **Records:** 1,331 BESS units across all US ISOs
- **ERCOT subset:** 155 operating units, 10,105 MW total
- **Key columns:** Plant Name, Nameplate Capacity (MW), Energy Capacity (MWh), Status, Balancing Authority, Pricing Node, Lat/Long, Technology, County

### Backtest Results (`results/`)
- `bess_backtest_summary_2024.csv` — Annual revenue per location (hubs + nodes, DA + RT)
- `sensitivity_rte.csv` — Revenue vs RTE sweep (80-95%)
- `sensitivity_duration.csv` — Revenue vs duration sweep (1-8h)
- `*.png` — Hub comparison, hub vs node, DA vs RT, sensitivity, dispatch profile charts
